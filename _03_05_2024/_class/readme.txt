




  1 Класс Event: Этот класс представляет событие с датой и описанием.
 Он реализует интерфейс Comparable<Event>, что позволяет сравнивать объекты Event.
  2 Метод compareTo: Этот метод используется для сравнения двух событий.
 Сначала он сравнивает даты событий. Если даты не равны, он возвращает результат сравнения дат.
 Если даты равны, он возвращает результат сравнения описаний событий.
 Это обеспечивает сортировку событий сначала по дате, а затем по описанию.
  3 Метод main: В этом методе создается массив событий.
 Затем этот массив сортируется с помощью метода Arrays.sort(),
 который использует метод compareTo для определения порядка сортировки.
 Наконец, все события в отсортированном массиве выводятся на экран.

 Таким образом, эта программа позволяет создавать события, сортировать их по дате и описанию,
 а затем выводить отсортированный список событий. Это достигается с помощью интерфейса Comparable,
 который позволяет определить естественный порядок сортировки для класса Event.

 compareTo - Comparable Сравнивает этот объект с указанным объектом для упорядочивания.
  Возвращает отрицательное целое число,
  ноль или положительное целое число,
  поскольку этот объект меньше, равен или больше указанного объекта.
  Разработчик должен гарантировать, что Signum(x.compareTo(y)) == -signum(y.compareTo(x)) для всех x и y.
  (Это подразумевает, что x.compareTo(y) должен генерировать исключение тогда и только тогда,
   когда y.compareTo(x) генерирует исключение.)
   Разработчик также должен гарантировать, что отношение является транзитивным:
    (x.compareTo(y) > 0 && y.compareTo(z) > 0)
    подразумевает x.compareTo(z) > 0.
    Наконец, разработчик должен гарантировать,
    что x.compareTo(y)==0 подразумевает,
    что Signum(x.compareTo(z)) == Signum( y.compareTo(z)), для всех z.
    Указывается:
    CompareTo в интерфейсе Comparable Параметры: o – объект для сравнения.
    Возвращает: отрицательное целое число, ноль или положительное целое число,
    поскольку этот объект меньше, равен или больше указанного объекта.

    Этот метод compareTo принадлежит классу Event и используется
    для сравнения текущего объекта Event (обозначенного как this) с другим объектом Event (обозначенным как other).

    int dateComparison = this.date.compareTo(other.date);
    - Эта строка сравнивает дату текущего события (this.date) с датой другого события (other.date).
    Метод compareTo для дат возвращает отрицательное число,
    если this.date раньше other.date, положительное число,
    если this.date позже other.date, и 0, если даты равны.
    if (dateComparison != 0) { return dateComparison; }
    - Если даты не равны (то есть dateComparison не равно 0),
    метод возвращает результат сравнения дат.
    Это означает, что события сортируются сначала по дате.
    else { return this.description.compareTo(other.description); }
    - Если даты равны (то есть dateComparison равно 0), метод сравнивает описания событий.
    Он возвращает результат сравнения описаний, что означает,
    что при равных датах события сортируются по описанию.

    Таким образом, этот метод обеспечивает сортировку событий сначала по дате,
    а затем по описанию. Это называется “естественным порядком” для объектов Event.
    Этот порядок используется при вызове Arrays.sort() или любого другого метода,
    который сортирует объекты Event.
